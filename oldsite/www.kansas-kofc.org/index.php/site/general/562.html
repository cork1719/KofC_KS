Athletic Events

The State Council sponsors a number of sporting events each fraternal year that are hosted by local councils across the entire state.

Councils wishing to host a state sponsored athletic event must submit their request in writing to the State Council Activity Director before January 22 of each year. It is better to submit the request earlier if possible.

The state sponsored athletic events are listed on the calendar on this website, and more information on each activity can be found by clicking here.
